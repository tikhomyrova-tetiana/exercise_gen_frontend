1. Clone the app

```
git clone git@github.com:YOUR_GITHUB_NAME/YOUR_PROJECT_NAME.git
```

2. cd into your project

```
cd YOUR_PROJECT_NAME
```

3. install dependencies

```
npm install
```

4. Start development server with npm start

```
npm start
```
